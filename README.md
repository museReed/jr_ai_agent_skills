## 繁體中文

### 這是什麼？

三個實用的 AI Agent Skills，讓你的 Claude Code / Codex CLI 更好用：

| Skill | 一句話說明 | 文件 |
|---|---|---|
| **Structured Questions** | 遇到多個方案時，AI 自動拆成選擇題問你，不替你決定 | [介紹](zh-Hant/structured-questions-skill.md) · [安裝](zh-Hant/structured-questions-install.md) |
| **Auto-Rename** | AI 自動幫每個 session 取名字，tab 不再全叫 "New chat" | [介紹](zh-Hant/auto-rename-skill.md) · [安裝](zh-Hant/auto-rename-install.md) |
| **Handoff** | Session 結束前自動寫交接文件，下次不用從頭來 | [介紹](zh-Hant/handoff-skill.md) · [安裝](zh-Hant/handoff-install.md) |

### 怎麼安裝？

把這個 repo clone 下來，然後讓 AI 讀安裝指南：

```bash
git clone https://github.com/museReed/jr_ai_agent_skills.git
```

**Claude Code：**
```
Read jr_ai_agent_skills/zh-Hant/structured-questions-install.md and execute Section A
```

**Codex CLI：**
```
Read jr_ai_agent_skills/zh-Hant/structured-questions-install.md and execute Section B
```

其他 skill 同理，換檔名即可。

### 適用工具

| 工具 | 支援 |
|---|---|
| Claude Code | 完整支援（skill + hook 自動觸發） |
| Codex CLI | 支援 skill，部分功能需手動觸發 |

## English

### What is this?

Three practical AI Agent Skills that make your Claude Code / Codex CLI more useful:

| Skill | Description | Docs |
|---|---|---|
| **Structured Questions** | When there are multiple options, AI breaks them into structured choices instead of deciding for you | [Guide](en/structured-questions-skill.md) · [Install](en/structured-questions-install.md) |
| **Auto-Rename** | AI automatically names each session so your tabs aren't all called "New chat" | [Guide](en/auto-rename-skill.md) · [Install](en/auto-rename-install.md) |
| **Handoff** | Automatically creates a handoff document before session ends so the next session can pick up seamlessly | [Guide](en/handoff-skill.md) · [Install](en/handoff-install.md) |

### How to install?

Clone this repo, then have AI read the install guide:

```bash
git clone https://github.com/museReed/jr_ai_agent_skills.git
```

**Claude Code:**
```
Read jr_ai_agent_skills/en/structured-questions-install.md and execute Section A
```

**Codex CLI:**
```
Read jr_ai_agent_skills/en/structured-questions-install.md and execute Section B
```

Same pattern for other skills — just swap the filename.

### Supported Tools

| Tool | Support |
|---|---|
| Claude Code | Full support (skill + hook auto-trigger) |
| Codex CLI | Skill supported, some features require manual trigger |
